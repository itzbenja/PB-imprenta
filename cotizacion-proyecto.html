¿Qué es este sistema?

Una aplicación web que reemplaza las planillas Excel de cotización por un sistema en tiempo real, accesible desde cualquier dispositivo. El vendedor ingresa las especificaciones del pedido y el sistema calcula automáticamente el costo de producción, margen de ganancia e IVA — en segundos.

Funcionalidades principales

🧮
Motor de cotización automático Calcula papel, procesos, planchas, laminado y terminaciones con precios reales desde la base de datos
📐
Visualizador geométrico de imposición Grafica en tiempo real el rendimiento del pliego para el operador de guillotina — muestra distribución, rotación y merma
💰
Margen de ganancia configurable Slider de 0–100% + merma de papel 5–20%, IVA 19% automático incluido en el total
📦
Terminaciones completas Laminado (Flyers, Tapas, Agendas), Anillado, Cosido, Pegado, Engomado, Prepicado, Corchete y Cinta
🔐
Panel de administración protegido Login con email/contraseña. CRUD completo de máquinas, insumos, procesos y precios
☁️
Precios en tiempo real El admin actualiza un precio en Firebase y todos los usuarios ven el cambio sin recargar la página
📋
Historial de cotizaciones Cada cotización se guarda automáticamente en la nube con fecha, tipo de producto y costos
🛡️
Seguridad en base de datos Reglas de Firestore: validación de campos, rangos y tipos. Solo datos válidos son aceptados

✓ Recomendación: Firebase
El sistema ya está 100% construido sobre Firebase. Migrar a Supabase o AWS requeriría reescribir la capa de datos completa (semanas de trabajo) sin ningún beneficio real para el tamaño de esta empresa. Firebase se publica en 30 minutos con firebase deploy.
